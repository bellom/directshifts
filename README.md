## Installation

1. Clone the project to your local directory

```
git clone https://github.com/bellom/directshifts
```

2. Run `bundle install`

3. Run `rails db:create`

4. Run `Rails db:migrate`

5. Run `rails s -p 3001`

6. Open another terminal

7. cd `client` 

8. Run `npm install`

9. Run `npm start`
